Speckle offers multiple ways to send and receive data — making it easy to fit into your existing workflows, no matter what tools you use.

Whether you're working in a desktop application, uploading files, or connecting cloud services, Speckle helps you centralize and manage your design and construction data with minimal friction.

## Here's how it works:

- **Speckle Connectors**  
  Our desktop plugins for tools like **Revit**, **Rhino**, **Grasshopper**, **AutoCAD**, **SketchUp**, and more let you publish and receive data directly within your design environment.

- **File Uploads**  
  Upload common 3D file formats like `.ifc`, `.obj`, and `.stl` through the **Speckle Web App** — no plugin required.

- **Third-Party Integrations**  
  Speckle connects with platforms such as:
  - **Shapediver** for business intelligence  
  - **Vectorworks** for project coordination  
  - **Karamba** for automating workflows  
  - Custom integrations using the [**Speckle API** or **SDKs**](http://localhost:3000/api-reference/introduction)

Each method supports key Speckle features like versioning, data filtering, and collaboration — helping your team stay connected across tools and stages of the project.
