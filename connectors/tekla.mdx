<Note>Tekla connector is publish-only.</Note>

<Info>Speckle currently supports Tekla versions: **2023, 2024 and 2025**.</Info>

## Getting Started

Welcome! This guide will walk you through how to **publish** your Tekla models using **Tekla connector**.

<Note> Before you begin, make sure you have a Speckle account and [installed Tekla Connector](http://localhost:3000/connectors/installation).</Note>

<Steps>
  <Step title="Open Tekla connector">
    1. Go to the **Applications & components** button in the ribbon.
    2. Search for **Speckle** and select it to open Tekla connector.
  </Step>

  <Step title="Publish to Speckle">
    1. In the Speckle connector, select **Publish**.
    2. Choose a [project]():
        - Select an existing project
        - or select **New Project** to create one
    3. Choose a [model]():
        - Select an existing model
        - or select **New Model** to create one
    4. In your Tekla viewport, select the elements you want to publish.
    5. Select **Publish**.
    
    Your model will be uploaded to Speckle and becomes available with the rest of your team.
  </Step>
</Steps>

<Tip> Check the video and steps in the [first steps](/get-started/quickstart.mdx) section to get started even faster!</Tip>
