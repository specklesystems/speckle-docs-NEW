## Getting Started

Welcome! This guide will walk you through how to **publish** your Revit models and **load** other models using **Revit connector**.

<Note>
  Before you begin, make sure you have a Speckle account and [installed Revit
  Connector](http://localhost:3000/connectors/installation).
</Note>

<Tabs>
  <Tab title="Step 1: Open Revit Connector">
    <Steps>
      <Step
        title="In Revit, select the Speckle tab in the ribbon."
        description="Open the Speckle connector in Revit."
      ></Step>
      <Step title="Select Speckle to open the Revit connector."></Step>
    </Steps>
  </Tab>
  <Tab title="Step 2: Publish to Speckle">
    <Steps>
      <Step
        title="In the Speckle connector, select Publish."
        description="Choose a project and model to publish your Revit elements."
      ></Step>
      <Step
        title="Select a project and model."
        description="You can either select an existing project/model or create a new one."
      ></Step>
      <Step
        title="In your Revit viewport, select the elements you want to publish."
        description="You can select multiple elements at once."
      ></Step>
      <Step title="Click Publish to upload your model to Speckle."></Step>
    </Steps>
  </Tab>
  <Tab title="Step 3: Load from Speckle">
    <Steps>
      <Step
        title="In the Speckle connector, select Load."
        description="Choose a project, model, and version to load."
      ></Step>
      <Step
        title="Select a project and model."
        description="You can either select an existing project/model or create a new one."
      ></Step>
      <Step
        title="Choose a version to load."
        description="Latest version is loaded by default. To load a specific version, select it from the UI."
      ></Step>
      <Step title="Click Load to load the selected model in Revit."></Step>
    </Steps>
  </Tab>
</Tabs>

<Tip>
