<Info>Speckle currently supports SketchUp versions: **2021, 2022, 2023, 2024, and 2025**. SketchUp connector works on **Windows** only.</Info>

## Getting Started

Welcome! This guide will walk you through how to **publish** your SketchUp models and **load** other models using **SketchUp connector**.

<Note> Before you begin, make sure you have a Speckle account and [installed SketchUp Connector](http://localhost:3000/connectors/installation).</Note>

<Steps>
  <Step title="Open SketchUp connector">
    1. Navigate to Extensions > Speckle.
    2. Select **Initialize Speckle** to open SketchUp connector.
  </Step>

  <Step title="Publish to Speckle">
    1. In the Speckle connector, select **Publish**.
    2. Choose a [project]():
        - Select an existing project
        - or select **New Project** to create one
    3. Choose a [model]():
        - Select an existing model
        - or select **New Model** to create one
    4. In your SketchUp viewport, select the elements you want to publish.
    5. Select **Publish**.
    
    Your model will be uploaded to Speckle and becomes available with the rest of your team.
  </Step>


    <Step title="Load from Speckle">
      1. In the Speckle connector, select **Load**.
    2. Select the [project]().
    3. Select the [model]().
    4. Choose a [version]():
          - Latest version is loaded by default.
          - To load a specific version, select it from the UI.
      5. Select **Load**.

      Selected model will be loaded in SketchUp.
    </Step>

</Steps>
