---
title: Installation
description: "How to download and install a Connector"
---

1. Create a Speckle <br/> Visit [speckle.systems](https://app.speckle.systems/) and create an account.

2. Download and Install  
After creating your account, visit [speckle.systems/connectors](https://app.speckle.systems/connectors) and download the Connector for your software (e.g., Rhino, AutoCAD, SketchUp, etc.).  


3. Install
Run the installer and follow the on-screen instructions.
<Warning>Please ensure that the target software is closed during the installation process.</Warning>
