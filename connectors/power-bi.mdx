<Info>Speckle currently supports Power BI versions: **2022, 2023, 2024, 2025, and 2026**.</Info>

## Getting Started

Welcome! This guide will walk you through how to **load** a model using **Power BI connector**.

<Note> Before you begin, make sure you have a Speckle account and [installed Power BI Connector](http://localhost:3000/connectors/installation).</Note>

<Steps>
    <Step title="Open Power BI connector">
    1. Select **Get Data**.
    2. In the search box, enter *Speckle*.
    3. Select **Connect to Speckle** and click **Connect**.
    </Step>

    <Step title="Load from Speckle">
    Paste the Model URL in the text field and select **OK**.
    </Step>

</Steps>

<Tip> Check the video and steps in the [first steps](/get-started/quickstart.mdx) section to get started even faster!</Tip>
