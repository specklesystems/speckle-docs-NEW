<Note>Navisworks connector is publish only.</Note>

<Info>Speckle currently supports Navisworks versions: **2020, 2021, 2022, 2023, 2024, 2025, and 2026**.</Info>

## Getting Started

Welcome! This guide will walk you through how to **publish** your Navisworks models using **Navisworks connector**.

<Note> Before you begin, make sure you have a Speckle account and [installed Navisworks Connector](http://localhost:3000/connectors/installation).</Note>

<Steps>
  <Step title="Open Navisworks connector">
    1. In Navisworks, select the **Speckle** tab in the ribbon.
    2. Select **Speckle** to open Navisworks connector.
  </Step>

  <Step title="Publish to Speckle">
    1. In the Speckle connector, select **Publish**.
    2. Choose a [project]():
        - Select an existing project
        - or select **New Project** to create one
    3. Choose a [model]():
        - Select an existing model
        - or select **New Model** to create one
    4. In your Navisworks viewport, select the elements you want to publish.
    5. Select **Publish**.
    
    Your model will be uploaded to Speckle and becomes available with the rest of your team.
  </Step>
</Steps>

<Tip> Check the video and steps in the [first steps](/get-started/quickstart.mdx) section to get started even faster!</Tip>
