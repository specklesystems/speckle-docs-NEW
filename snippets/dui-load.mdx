<Step title="Load from Speckle">
    1. In the Speckle connector, select **Load**.
    ![title](/images/connectors/dui_load_1.jpg)
    2. Select the [project]().
    ![title](/images/connectors/dui_load_2.jpg)
    3. Select the [model]().
    ![title](/images/connectors/dui_load_3.jpg)
    4. Choose a [version]():
          - Latest version is loaded by default.
          - To load a specific version, select it from the UI.
    ![title](/images/connectors/dui_load_4.jpg)
    5. Select **Load**.
    ![title](/images/connectors/dui_load_5.jpg)

    Your selected model will be loaded in {connector}.
