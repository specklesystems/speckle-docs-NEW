  <Step title="Publish to Speckle">
    <Frame>
    <img src="/images/connectors/dui_publish_1.jpg" alt="Publish" />
    </Frame>
    1. In the Speckle connector, select **Publish**.
    2. Choose a [project]():
        - Select an existing project
        - or select **New Project** to create one
    <Frame>
    <img src="/images/connectors/dui_publish_2.jpg" alt="Publish" />
    </Frame>
    3. Choose a [model]():
        - Select an existing model
        - or select **New Model** to create one
    <Frame>
    <img src="/images/connectors/dui_publish_3.jpg" alt="Publish" />
    </Frame>
    4. In your application viewport, select the elements you want to publish.
    <Frame>
    <img src="/images/connectors/dui_publish_4.jpg" alt="Publish" />
    </Frame>
    5. Select **Publish**.
    <Frame>
    <img src="/images/connectors/dui_publish_5.jpg" alt="Publish" />
    </Frame>
    
    Your model will be uploaded to Speckle and becomes available to the rest of your team.
  </Step>
