## Environment Files

Copy the example files to configure your environment:

```bash
cp packages/server/.env.example packages/server/.env
cp packages/server/.env.test-example packages/server/.env.test
cp packages/frontend-2/.env.example packages/frontend-2/.env
```

## Key Variables

### Server Configuration
```env
SPECKLE_SERVER_URL=http://localhost:3000
NODE_ENV=development
PORT=3000
```

### Database
```env
POSTGRES_DB=speckle
POSTGRES_USER=speckle
POSTGRES_PASSWORD=speckle
POSTGRES_HOST=localhost
POSTGRES_PORT=5432
```

### Storage
```env
REDIS_URL=redis://localhost:6379
MINIO_ROOT_USER=minioadmin
MINIO_ROOT_PASSWORD=minioadmin
MINIO_ENDPOINT=localhost
MINIO_PORT=9000
```

### Security
```env
JWT_SECRET=your-jwt-secret
SESSION_SECRET=your-session-secret
ENCRYPTION_KEY=your-encryption-key
```

## Production

For production, ensure you:
- Use strong secrets for JWT_SECRET, SESSION_SECRET, and ENCRYPTION_KEY
- Set NODE_ENV=production
- Configure proper email settings
- Use secure database credentials

<Note>
For complete environment variable reference, see the `.env.example` files in the [Speckle server repository](https://github.com/specklesystems/speckle-server).
</Note>
